# Persiapan Python di Neovim(NvChad)

Paduan ini menjelaskan cara setup environment **Python** di **Neovim** menggunakan **NvChad** atau menggunakan **[Dotfiles](https://github.com/gh0st4n/my-ide)**.

## 1. Requirements
- **Neovim**(include NvChad)
- **Python**
- **Alacritty**(Opsional, tapi Rekomendasi atau Terminal lainnya)
- **Nerd Font**  → *FiraCode Nerd Font*
- **Tree-Sitter CLI**
- **Ripgrep**

### Python Package Manager

Gunakan **UV** (lebih cepat dari pip):

#### Install via curl:

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

#### Install via wget:

```bash
wget -qO- https://astral.sh/uv/install.sh | sh
```

## 2. Setup NvChad

Jika belum terinstall:

```bash
git clone https://github.com/NvChad/starter ~/.config/nvim && nvim
```

## 3. Setup LSP (Python)

### Install Ty

Di dalam Neovim:

```vim
:MasonInstall ty
```

### Konfigurasi LSP

Edit:

```lua
~/.config/nvim/lua/config/lspconfig.lua
```

```lua
local servers = { "ty" }
```

## 4. Setup Tree-Sitter

Install parser Python:

```
:TSInstall python
```

## 5. Python Tools (Formatter, Linter, Debugger)

Install menggunakan pip atau uv:

```bash
pip install black ruff mypy debugpy
```

atau:

```bash
uv pip install black ruff mypy debugpy
```

### Fungsi Tools

* **black** → formatter kode
* **ruff** → linter cepat
* **mypy** → type checker
* **debugpy** → debugging

## 6. Update Environment

```
:Lazy sync
:MasonUpdate
```

## 7. Auto Format (Opsional tapi Disarankan)

Tambahkan ke:

```
~/.config/nvim/lua/custom/autocmds.lua
```

```lua
vim.api.nvim_create_autocmd("BufWritePre", {
  pattern = "*.py",
  callback = function()
    vim.lsp.buf.format()
  end,
})
```

## 8. Troubleshooting

### LSP tidak jalan

```
:LspInfo
```

Jika tidak muncul:

* Pastikan `ty` terinstall

### Tree-sitter error

```
:TSUpdate
```

### NvChad bermasalah

```bash
rm -rf ~/.local/share/nvim
rm -rf ~/.cache/nvim
nvim
```
